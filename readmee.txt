# 📦 Mini WMS (Warehouse Management System)

A full-stack Mini Warehouse Management System built using **Angular** and **.NET** to manage inventory operations efficiently. This project demonstrates real-world implementation of stock tracking with clean UI and proper API integration.

---

## 🚀 Features

* 📥 **Stock Inward Management** – Add incoming inventory
* 📤 **Stock Outward Management** – Track outgoing items
* 📊 **Transaction History** – View all inward & outward records
* ✅ **Form Validations** – Ensures accurate data entry
* 🔗 **API Integration** – Angular frontend connected with .NET backend
* 🎯 **User-Friendly UI** – Clean and responsive design

---

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/mini-wms-angular-dotnet.git
cd mini-wms-angular-dotnet
```

---

### 2️⃣ Backend Setup (.NET)

* Open backend project in Visual Studio
* Update database connection string in `appsettings.json`
* Run the API

---

### 3️⃣ Frontend Setup (Angular)

```bash
cd frontend
npm install
ng serve
```

* Navigate to: `http://localhost:4200/`
